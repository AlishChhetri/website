Software Developer

Computer and Information Science Department, Allegheny College | August 2023 - May 2024

Role Overview

As a Software Developer in the Computer and Information Science Department at Allegheny College, I led the development of Chasten, a Python-based static analysis tool. This role involved advanced programming, tool design, and integration of modern AI technologies to create a more intuitive approach to code analysis.

Key Accomplishments

  • Developed Chasten, a static analysis tool that uses XPath expressions to detect Abstract Syntax Tree (AST) patterns in code
  • Created a system enabling custom linting rules without requiring complex frameworks or relying on imprecise regex pattern matching
  • Led the integration of Large Language Models (LLMs) for natural language-to-XPath query conversion, significantly optimizing development time
  • Worked with faculty and students to refine the tool based on real-world programming education use cases

Technical Skills Applied

  • Python Development: Designed and implemented the core tool architecture in Python
  • AST Analysis: Created algorithms to parse, analyze, and query code syntax trees
  • XML/XPath: Implemented XML transformations and XPath query systems
  • LLM Integration: Developed interfaces to leverage LLM capabilities for code analysis
  • Testing: Implemented comprehensive testing suite to ensure tool reliability

Impact and Results

The development of Chasten has had significant impact on programming education and code quality assessment:

  • Provided faculty and students with a more intuitive way to detect and understand code patterns
  • Reduced the learning curve for creating custom linting rules by enabling natural language descriptions
  • Improved the precision of code pattern detection compared to traditional regex-based approaches
  • Created a foundation for future research in code analysis and educational technology
